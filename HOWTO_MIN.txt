1) Drag ALL files into Xcode and tick Target Membership → AllowanceAlley.
2) In AllowanceAlleyApp.swift inject once:
   @StateObject private var familyStore = FamilyStore.demo() // or real
   @StateObject private var ledger = AALedgerStore()
   ContentView()
     .environmentObject(familyStore)
     .environmentObject(ledger)
3) Forward your existing tabs:
   - In ChoresView.swift:   `var body: some View { ChoresV2View() }`
   - In RewardsView.swift:  `var body: some View { RewardsV2View() }` (or RewardsListView.swift)
4) Clean Build Folder (Shift+Cmd+K) and Run.
