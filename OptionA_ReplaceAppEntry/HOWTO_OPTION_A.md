# Option A — Replace App Entry
1) Drag these two files into Xcode. When prompted for `AllowanceAlleyApp.swift`, choose **Replace**.
2) Ensure **Target Membership → AllowanceAlley** is checked.
3) Product → Clean Build Folder (⇧⌘K) → Run.

If `FamilyStore()` fails to compile, change it to `FamilyStore.demo()` and run again.
