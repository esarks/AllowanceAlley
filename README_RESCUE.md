# AllowanceAlley — Rescue Pack (Zero Dependencies)

Use this to **force visible functionality** even if environment objects or routers are miswired.

- **Option A** replaces your app entry to show a standalone, self-contained Dashboard/Chores/Rewards.
- **Option B** lets you add the same Rescue view as an extra tab without touching your @main.

If you do Option A and still don't see changes, you're on the wrong scheme/target.
